## Theme Options

You customize the appearance of your Gusronk instance to align with branding. Select from predefined themes or create custom ones using CSS variables.

<Callout uid="0ef8c13d-2f60-4467-bc11-454c51bdaeed" kind="alert" collapsed="false">
  Test custom themes in preview mode before applying.
</Callout>

Access theme settings from the space admin panel. Choose colors, fonts, and logos. Gusronk supports light and dark modes automatically.

<Tabs uid="fbe51feb-768a-4f39-882f-fb8014540453">
  <Tab title="Prebuilt Themes" icon="palette">
    Pick from modern, classic, or minimal styles.
  </Tab>

  <Tab title="Custom CSS" icon="code">
    Override variables like `--primary-color`.

    ```css
    :root {
      --primary-color: #3B82F6;
      --font-family: 'Inter', sans-serif;
    }
    ```
  </Tab>
</Tabs>

Apply changes site-wide or per space.

## Layout Customization

Adjust doc layouts for better readability. Enable sidebars, tables of contents, or full-width views.

<Steps uid="bdb26446-14f2-4395-94b0-1eb8199e8e32">
  <Step title="Enable TOC" icon="list" title-type="p">
    Toggle in doc settings for automatic generation.
  </Step>

  <Step title="Sidebar Widgets" icon="sidebar" title-type="p">
    Add search, recent docs, or custom links.

    ```javascript
    // Widget configuration example
    const widgets = [
      { type: 'search', position: 'left' },
      { type: 'recent', limit: 5 }
    ];
    ```
  </Step>

  <Step title="Embed Components" icon="plus" title-type="p">
    Insert custom embeds like calendars or forms.
  </Step>
</Steps>

## Integrations

Connect Gusronk to external services for enhanced functionality. Supported integrations include Slack, GitHub, and Google Drive.

<CodeGroup uid="625d814a-95c7-4d38-9921-b09e651cb263" show-lines="true" tabs={[]}>
  ```javascript
  // Slack webhook setup
  const slackConfig = {
    webhook: 'https://hooks.slack.com/services/T00000000/B00000000/XXXXXXXXXXXXXXXXXXXXXXXX',
    channel: '#docs'
  };
  ```

  ```yaml
  # GitHub integration
  repo: owner/project
  branch: main
  webhook: https://gusronk.com/webhook/github
  ```
</CodeGroup>

You automate notifications or sync content. API keys secure connections.
