## Overview

Gusronk provides a centralized space for organizing and managing your project documentation. You create, edit, and collaborate on docs efficiently. Access features like version control and search to streamline your workflow.

## Getting Started Steps

Follow these steps to set up your first documentation space.

<Steps>
  <Step title="Create Account" icon="user">
    Sign up on the Gusronk dashboard.
  </Step>
  <Step title="New Space" icon="folder">
    Click the new space button and name it.
    ```bash
    # No CLI needed, but here's a mock for integration
gusronk create space my-project
    ```
  </Step>
  <Step title="Add Doc" icon="file-text">
    Within the space, add your first document.
  </Step>
</Steps>
